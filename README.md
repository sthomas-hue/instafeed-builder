# InstaFeed Builder Pro

A beautiful, customizable Instagram feed generator. Create embeddable feeds for your website (especially Webflow).

### Features
- Live preview
- Multiple layouts (Grid, Masonry, Carousel)
- Color picker
- Hover effects + Lightbox
- Save multiple widgets
- Auto-updates every 60 minutes
- One-click Webflow embed code

### How to Use
1. Enter an Instagram username
2. Customize layout, colors, etc.
3. Click **Generate Preview**
4. Click **Get Webflow Embed Code**
